# binary-clock
A 24 hour binary clock with 2 time setting buttons. To enter "time setting" mode and toggle between seconds/minutes/hours use the red button, and to set the value use the green button.

*BOM*:
* 17 LEDs
* 2 tactile buttons
* 2 1kOhm resistors
* 1 breadboard
* Arduino nano/uno
