# binary-clock
A 24 hour binary clock with time setting buttons.

*BOM*:
* 17 LEDs
* 17 100Ohm resistors
* 2 tactile buttons
* 2 1kOhm resistors
* 1 breadboard
* Arduino nano/uno
* Jumper wires
